#Palavras Embaralhadas
Projeto final do curso de Programação Orientada a Objetos do Coursera ministrado pelo ITA (Instituto Tecnológico de Aeronáutica)


O projeto consiste basicamente em criar um jogo cujo o objetivo é adivinhar qual é a palavra que é mostrada com as letras trocadas.

Para fazê-lo funcionar, foram criados dois algoritmos de embaralhamento - dificuldades normal e fácil - que são chamados aleatoriamente a cada novo início de partida. 

As palavras são extraídas de um arquivo (XML ou TXT) gerado dinamicamente.
